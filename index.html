Explanation 1

They're Kind of Like Loops


							/* ===== LOOP ===== */
							while (true) {
							  console.log("I'm a loop!!!");
							}

							/* ===== RECURSION ===== */
							function recursion() {
							  console.log("I'm a recursive function!!!")
							  recursion();
							}

							recursion(); /* function call */

Explanation 2

They're Kind of Like Function Call Arguments


							/* ===== FUNCTION CALL ARGUMENTS ===== */
							function add(a, b) { return a + b; }
							
							add(1, add(1, 1));
							/* becomes add(1, 2) */
							/* then becomes 3 */
						

							/* ===== RECURSION ===== */
							function recursion() {
							  if (3RD RECURSIVE CALL) {
							    /* STOP! */
							  } else {
							    recursion();
							  }
							}
							
							recursion(); /* recursive function call */

Reasons not to:
